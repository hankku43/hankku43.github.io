網頁設計

說明

此網頁就是我對於網頁開發工程的成果展示，在此處分享一些關於開發網頁的技術心得。

技術心得

本履歷網站主要參考的社群網站104求職網的履歷樣式、和複數社群網站的個人主頁樣式， 為了完成圖片層層交疊的效果，善用position: absolute;和z-index的設定可以簡潔快速地達成想要的排版， 然而有一些效果不是很直接能在CSS中找到可以設定調整的屬性參數，像是同時具有border-radius和漸層顏色的邊框； 或是文字的內陰影。然而經過我搜尋網路上前輩的技術分享部落格，成功用了特別的方式， 在網頁上方的金屬色字牌實現了前面提到的兩種效果，真的非常敬佩兩位想這樣子方式的前輩。 想要同時擁有border-radius和漸層顏色的邊框，用到的是2個交疊background-image屬性，藉由background-clip 和background-origin的控制讓後層的背景邊界大於上層，產生看起來像是邊框的效果。 而文字的內陰影效果則是利用帶有透明度顏色的字，搭配在下層的黑色陰影和同背景顏色並稍微偏移的陰影， 藉由同背景顏色且稍微偏移的陰影將黑色陰影不必要之處蓋住，產生類似內陰影的效果。

3D物件設計──OpanSCAD

說明

就讀研究所時，指導教授請我以角鋼廢料和木板製作一雙層鐵架供儀器置放，角鋼組裝的重點之一在於鎖孔的相對位置有無對齊， 在沒有設計規劃的情況下容易浪費材料，因此我在材料非常有限的情況下決定以電腦輔助設計的方式製作模型。 電腦輔助設計（Computer Aided Design, CAD）是指運用電腦軟體製作並類比實物設計，展現新開發商品的外型、結構、色彩、質感等特色的過程。 而OpanSCAD是一供程式設計師進行實體建模的軟體工具，以程式語言的邏輯和參數化的設計，進行3D物品的建模。

作品──角鋼雙層架

步驟一

角鋼是一剖面L型的長鐵條，L型兩個平面上規律佈滿著不同樣式的鎖孔，鎖孔的排列方式具有一定的規律， 我測量了鎖孔重複的長度，並繪製出最小重複單位的角鋼單元。 
R=4.5;                      //鎖孔半徑
T=1.5;                      //厚度
ux=50;                      //x軸長
uy=76;                      //y軸長
uz=32;                      //z軸長
module unit(m){             //新增模型 參數m為鎖孔相對位置起始值
difference()                //減法加工
    {
    cube([ux,uy,uz]);       //原塊    
    translate([T,0,T])      //位移
        cube([ux-T,uy,uz-T]);     //削減  
    translate([38,uy/2+m,0])
        linear_extrude(T)         //線性方向增厚
            circle(R);            //長邊中間圓洞         
    linear_extrude(T)             //長邊前橢圓洞
        hull() {                  //兩形狀線性連接 
            translate([38,uy/2-21.5+m,0]) circle(R);
            translate([38,uy/2-16.5+m,0]) circle(R);        
            } 
    linear_extrude(T)             //長邊後橢圓洞
        hull() {                  
            translate([38,uy/2+16.5+m,0]) circle(R);
            translate([38,uy/2+21.5+m,0]) circle(R);        
            }     
    linear_extrude(T)             //大洞
        hull() {                  
            translate([21,uy/2-21.5+m,0]) circle(R);
            translate([21,uy/2+21.5+m,0]) circle(R);        
            }         
    linear_extrude(T)             //長邊前直洞
        hull() {                  
            translate([38,0+m,0]) circle(R);        
            translate([34,0+m,0]) circle(R);                
            }        
    linear_extrude(T)             //長邊後直洞
        hull() {
            translate([38,uy+m,0]) circle(R);
            translate([34,uy+m,0]) circle(R);               
            }
    translate([0,uy/2+m,20])      //短邊中間圓洞
        rotate([0,90,0]){         //對Y軸旋轉                          
            linear_extrude(T)
            circle(R);}                                        
    translate([0,uy/2-21.5+m,20]) //短邊前橢圓洞
        rotate([0,90,0]){
            linear_extrude(T)
                hull() {
                translate([0,0,0]) circle(R);
                translate([0,5,0]) circle(R);
                }
        }                                                   
    translate([0,uy/2+16.5+m,20])  //短邊後橢圓洞
        rotate([0,90,0]){
            linear_extrude(T)
                hull() {
                translate([0,0,0]) circle(R);
                translate([0,5,0]) circle(R);
                }
        }                                                   
    translate([0,0+m,20])         //短邊前直洞
        rotate([0,90,0]){
            linear_extrude(T)
                hull() {
                translate([0,0,0]) circle(R);
                translate([5,0,0]) circle(R);
                } 
        }                                                   
    translate([0,uy+m,20])        //短邊後直洞
        rotate([0,90,0]){
            linear_extrude(T)
                hull() {
                translate([0,0,0]) circle(R);
                translate([5,0,0]) circle(R);
                } 
        }                                                       
    if (uy/2+16.5+m+R>uy){                //以m值判斷鎖孔循環                          
        linear_extrude(T)                 //長邊後橢圓洞循環
            hull() {
            translate([38,uy/2+16.5+m-uy,0]) circle(R);
            translate([38,uy/2+21.5+m-uy,0]) circle(R);     
            }
        linear_extrude(T)                 //大洞循環
            hull() {
                translate([21,uy/2-21.5+m-uy,0]) circle(R);
                translate([21,uy/2+21.5+m-uy,0]) circle(R); 
            }  
        translate([0,uy/2+16.5+m-uy,20])  //短邊後橢圓洞循環
            rotate([0,90,0]){
                linear_extrude(T)
                    hull() {
                        translate([0,0,0]) circle(R);
                        translate([0,5,0]) circle(R);       
                        }
                }
        }        
    if (uy/2+m+R>uy){
        translate([38,uy/2+m-uy,0])       //長邊中間圓洞循環
            linear_extrude(T) circle(R);                      
        translate([0,uy/2+m-uy,20])       //短邊中間圓洞循環
            rotate([0,90,0]){
                linear_extrude(T)
                circle(R);}                                   
        }
    }                                     //單位角鋼設定
}                                         //模塊設定

步驟二

有了單元角鋼，將其複製延長成想要的長度，並旋轉到對應角度，便可以微調參數m檢查各個鎖孔有無對齊，從底下影片中可以清楚看到角鋼交會處的鎖孔皆是相通的。 
module polynumber(num) {        //建立具有num個角鋼單元長度的角鋼條模型
    for (i = [0 : num-1])
    translate([ 0, uy*i, 0 ]) children();
}
module coner (x,y,H,m) {        //建立角鋼架4角直立柱模型
    rotate([90,0,90])polynumber(H){ unit(m);}
    translate([x,0,0]) mirror([1,0,0]) rotate([90,0,90])polynumber(H){ unit(m);}
    translate([0,y,0]) mirror([0,1,0]) rotate([90,0,90])polynumber(H){ unit(m);}
    translate([x,y,0]) rotate([90,0,270])polynumber(H){ unit(m);}
}
module frame(xn,yn,xm,ym){      //建立角鋼橫條模型
    translate([T+T,T,ux]) mirror([1,0,0])   rotate([270,90,0]) polynumber(xn) unit(xm);
    translate([T+T,uy*yn-T,ux]) rotate([0,90,270]) polynumber(xn) unit(xm);
    difference(){
        translate([T,T,ux+T])rotate([0,90,0])    polynumber(yn) unit(ym);
        translate([T,T+(yn*uy)-T-T,ux+T])rotate([0,90,0])cube([ux,uy,uz]);
        }
    difference(){
        translate([uy*xn+T+T+T,T,ux+T])mirror([1,0,0])rotate([0,90,0])polynumber(yn) unit(ym);
        translate([uy*xn+T+T+T,T+(yn*uy)-T-T,ux+T])mirror([1,0,0])rotate([0,90,0])cube([ux,uy,uz]);
        }
    }
    
color("Teal")coner(uy*4+T+T+T+T,T+T+uy*7-T-T+0.001,7,-8.25);    //繪製4角直立柱
color("Teal")frame(4,7,0,-1.5);                                 //繪製底部橫條
color("Teal")translate([0,0,5*uy+9+7.5+0.75]) frame(4,7,0,-1.5);//繪製上層橫條
